<h3><i> ◎ 오디오 태그 </i></h3>

```html
<!DOCTYPE html>
<html>
<head>
	<meta charset="utf-8">
	<meta name="viewport" content="width=device-width, initial-scale=1">
	<title> 오디오 파일 </title>
</head>
<body>
	<audio src="evocation.mp3" autoplay controls loop></audio>
</body>
</html>
```

- MP3, WAV, Ogg 등 오디오 파일 재생할 때 사용
- 태그 속성
 |   속성   |             설명             |
 | :------: | :--------------------------: |
 |   src    |          파일 경로           |
 | controls | 화면 플레이어 표시 여부 지정 |
 | autoplay |        자동 시작 지정        |
 |   loop   |        무한 반복 지정        |


<br/><br/>

- 비디오 파일 삽입 떄 사용
- 태그 속성
 |   속성   |             설명             |
 | :------: | :--------------------------: |
 |   src    |          파일 경로           |
 | controls | 화면 플레이어 표시 여부 지정 |
 | autoplay |        자동 시작 지정        |
 |   loop   |        무한 반복 지정        |

 |  height  |       동영상 높이 지정       |
 |  source  | 확장자에 따른 파일 type 지정 |
