## 이미지・영상 태그

```html
<!DOCTYPE html>
<html>
<head>
	<meta charset="utf-8">
	<meta name="viewport" content="width=device-width, initial-scale=1">
	<title>이미지 태그의 속성</title>
</head>
<body>
	<h3> 해당화 </h3>
		<img src="flower2.jpg" title="해당화(원본)">
		<img src="flower2.jpg" width="300" height="212" title="해당화(300X212)">
		<img src="flower2.jpg" width="150" title="해당화(가로:150, 세로:자동계산)">
</body>
</html>
```

1. `<img>` 태그
  - 이미지 파일
    - `PNG`
      - 파일 확장자 : `.png`
      - 가장 많이 사용하는 파일
    
    - `JPG`
      - 파일 확장자 : `.jpg`
      - 사진 이미지 저장
    
    - `GIF`
      - 파일 확장자 : `.gif`
      - 애니메이션용
  
  - 태그 속성
    - `src` : 이미지 파일 경로 + 이미지 이름
      - 절대 경로 : 이미지 웹브라우저 주소
      - 상대 경로 : 컴퓨터 안의 파일 경로
    
    - `title` : 이미지 설명 (브라우저에서의 이미지 이름)
    - `width` : 이미지 너비
    - `height` : 이미지 높이


2. `<embed>` 태그
  - 외부 파일을 재생하기 위해 사용
  - 사진이나 동영상이 아닌 경우 사용

```html
<!DOCTYPE html>
<html>
<head>
	<meta charset="utf-8">
	<meta name="viewport" content="width=device-width, initial-scale=1">
	<title></title>
</head>
<body>
	<embed src="전자책_Do it! HTML+CSS+자바스크립트 웹 표준의 정석 - 특별부록.pdf" width="1920" height="2080"></embed>
</body>
</html>
```

3. `<audio>` 태그
  - MP3, WAV, Ogg 등 오디오 파일 재생할 때 사용
  - 태그 속성
    - src : 삽입하는 오디오 파일 경로
    - controls : 화면 플레이어 표시 여부 지정
    - autoplay : 자동 시작 지정
    - loop : 무한 반복 지정

```html
<!DOCTYPE html>
<html>
<head>
	<meta charset="utf-8">
	<meta name="viewport" content="width=device-width, initial-scale=1">
	<title></title>
</head>
<body>
	<audio src="evocation.mp3" autoplay controls loop></audio>
</body>
</html>
```

4. `<video>` 태그
  - 비디오 파일 삽입 떄 사용
   - 태그 속성
     - src : 삽입하는 비디오 파일 경로
     - controls : 화면 플레이어 표시 여부 지정
     - autoplay : 자동 시작 지정
     - loop : 무한 반복 지정
     - width : 동영상 너비 지정
     - height : 동영상 높이 지정
     - source : 확장자에 따른 파일 type 지정

```html
<!DOCTYPE html>
<html>
<head>
	<meta charset="utf-8">
	<meta name="viewport" content="width=device-width, initial-scale=1">
	<title></title>
</head>
<body>
	<video width="320" height="240" controls autoplay>
		<source src="movie.mp4" type="video/mp4">
		<source src="movie.wepm" type="video/wepm">
		<source src="movie.ogg" type="video/ogg">
	</video>
</body>
</html>
```


## 테이블 태그

- `<table>` : 테이블 전체를 생성
- `<tr>` : 행을 나타낼 때 사용
- 열을 나타낼 때 사용 태그
  - `<th>` : 표의 머리 부분
  - `<td>` : 표의 몸통 부분
  
- 속성 태그
  - colspan : 열 병합
  - rowspan : 행 병합

```html
<!DOCTYPE html>
<html>
<head>
	<meta charset="utf-8">
	<meta name="viewport" content="width=device-width, initial-scale=1">
	<style>
		table, th, tr, td{
			border:solid 1px black;
			border-collapse: collapse;
			padding: 8px;
		}
	</style>
</head>
<body>
	<h3>오늘의 날씨</h3>
	<table>
		<tr>
			<th>지역</th>
			<th>현재기온</th>
			<th colspan="2">불쾌지수/습도(%)</th>
			<th>풍속(m/s)</th>
		</tr>
		<tr>
			<td rowspan="2">서울/경기</td>
			<td>23</td>
			<td>60</td>
			<td>80</td>
			<td>4.7</td>
		</tr>
		<tr>
			<td>20</td>
			<td>60</td>
			<td>80</td>
			<td>5.0</td>
		</tr>
		<tr>
			<td>제주도</td>
			<td>21</td>
			<td>65</td>
			<td>85</td>
			<td>3.6</td>
		</tr>
	</table>
</body>
</html>
```
