- <span style="color:chartreuse"> 연산 </span>
  - <span style="color:yellowgreen"> SUM( ) </span>
    - 합계
    - SUM( 인수1, 인수2, ... , 인수n ) : 인수들끼리의 합계
  - <span style="color:palegreen"> AVERAGE( ) </span>
    - 평균
    - AVERAGE( 인수1, 인수2, ... , 인수n ) : 인수들끼리의 평균

<br/>

  1. 급여총액
       - 급여총액 = 기본급여 + 상여금
       - SUM(기본급여, 상여금)
         - 옆으로 드래그하면 상대참조되서 다른 값들의 급여총액을 구할 수 있음

  2. 총액
        - 10월~3월까지 모든 급여의 합
        - SUM(10월 급여, 11월 급여, 12월 급여, 1월 급여, 2월 급여, 3월 급여)
          - 밑으로 드래그하면 상대참조되서 다른 값들의 총액을 구할 수 있음
  
  3. 평균
        - 10월~3월까지 모든 급여의 평균
        - AVERAGE(10월 급여, 11월 급여, 12월 급여, 1월 급여, 2월 급여, 3월 급여)
          - 밑으로 드래그하면 상대참조되서 다른 값들의 평균을 구할 수 있음

- <span style="color:dodgerblue"> 날짜 </span>
  - <span style="color:skyblue"> YEAR( ), MONTH( ), DAY( ), DATE( ) </span>
    1. YEAR( 인수1 ) : 년도만 출력
    2. MONTH( 인수1 ) : 월만 출력
    3. DAY( 인수1 ) : 일만 출력
    4. DATE( year, month, day ) : 날짜 합치기 (year-month-day 형식으로 출력)
  
  - <span style="color:paleturquoise"> TODAY( ) </span>
    - 해당 날짜 ( 사용한 날의 날짜 )

<br/>

  1. 작성일
       - 현재날짜를 표시하는 것이 조건이기에 TODAY( ) 사용
       - TODAY( ) 만 치고 Enter
       - 수식 - 날짜및시간 - TODAY( ) 클릭
  
  2. 승진일
        - 입사후 3년 하고도 입사월 5일자가 조건이고, 전체 날짜를 작성해야 하기에 DATE() 사용
        - year 인수
          - 입사후 3년이 조건이므로 YEAR( )를 사용해서 입사일 년도를 가지고 옴
          - YEAR( 입사일 ) + 3
        - month 인수
          - 입사월이니까 MONTH( )를 사용해서 입사일 월을 가지고 옴
          - MONTH( 입사일 )
        - day 인수
          - 5일자니까 가지고 올 필요없이 그냥 5 작성
        - DATE( YEAR( 입사일 )+3, MONTH( 입사일 ), 5 )
 
  3. 입사일・승진일
        - 입사일, 승진일 둘 다 똑같음
        - YEAR( )을 써서 년도를 가지고 오고, MONTH( )를 써서 월을, DAY( )를 써서 일을 가지고 오면 됨
        - YEAR( 입사일/승진일 ), MONTH( 입사일/승진일 ), DAY( 입사일/승진일 )

- <span style="color:dodgerblue"> 시간 </span>
  - <span style="color:skyblue"> HOUR( ), MINUTE( ), SECOND( ), TIME( ) </span>
    1. HOUR( 인수1 ) : 시간만 출력
    2. MINUTE( 인수1 ) : 분만 출력
    3. SECOND( 인수1 ) : 초만 출력
    4. TIME( hour, minute, second ) : 시간 합치기 (hour:minute:second 형식으로 출력)

<br/>

  1. 시간
       - HOUR( )을 써서 몇 시에 출근했는지 가지고 옴
       - HOUR( 출근시간 )

  2. 분
       - MINUTE( )을 써서 몇 분에 출근했는지 가지고 옴
       - MINUTE( 출근 시간 )

  3. 초
       - SECOND( )을 써서 몇 초에 출근했는지 가지고 옴
       - SECOND( 출근 시간 )

  4. 퇴근시간
       - 퇴근은 출근 후 8시간 30분이니까 TIME( )을 사용
         - hour 인수
           - 출근 후 8시간 후니까 HOUR( )을 써서 출근시간을 가지고 옴
           - HOUR( 출근시간 ) + 8
         - MINUTE 인수
           - 출근 후 30분 후니까 MINUTE( )을 써서 출근시간을 가지고 옴
           - MINUTE( 출근시간 ) + 30
         - second 인수
           - 출근 후 8시 30분 후니까 초는 그대라서 SECOND( )로 초만 가지고 옴
           - SECOND( 출근시간 )

- <span style="color:dodgerblue"> 조건문 </span>
  - <span style="color:turquoise"> IF() </span>
    - IF문 사용 개수 = ( 조건의 개수 - 1 )
    - IF( 조건, VT, VF )
      - 조건에 대해 값이 참일 경우 VT 인수값이 출력, 거짓일 경우 VF 인수값이 출력

    1. 영업상태
        - 조건: 평균이상인 경우 "안정", 평균미만인 경우 "개선"
          - IF( )는 1번만 사용
          - 전체평균 매출액은 움직이면 안되기 때문에 고정시켜 줘야함
          - 조건 인수
            - 1사분기 평균매출액 >= 전체평균 매출액
          - VT 인수 : 안정, VF 인수 : 개선
          - IF( 1사분기 평균매출액 >= $(전체평균 매출액), "안정", "개선" )

    2. 등급
        - 조건 : 평균매출액 기준 70,000 이상이면 "A", 70,000미만~50,000이상이면 "B", 50,000미만이면 "C"로 표기
        - 조건 총 3개니까 IF( )는 2번 사용
        - 조건1 인수
          - 1사분기 평균매출액 >= 70000
        - VT 인수 : "A"
        - VF 인수
          - "A"가 아닐 때는 "B" 아니면 "C"니까 여기서 IF( )문 사용
          - 조건2 인수
            - 1사분기 평균매출액 >= 50000
          - VT 인수 : "B", VF 인수 : "C"
        - IF( 1사분기 평균매출액 >= 70000, "A", IF( 1사분기 평균매출액 >= 50000, "B", "C" ) )

    3. 승인여부
       - 조건 : 3월 매출액이 50,000 이상이거나 총매출액이 150,000 이상인 경우 승인, 아니면 미승인
       - 조건이 총 2개니까 IF( )는 1번만 사용
       - 조건 인수
         - 3월 매출액이 50,000 이상'이거나' 총매출액이 150,000 이상
         - <span style="color:paleturquoise"> OR( ) </span> 사용
           - OR( 인수1, 인수2, ... , 인수n )
           - 인수 중 하나라도 부합하면 참
         - OR( 3월 매출액 >= 50000, 총매출액 >= 150000)
       - VT 인수 : "승인", VF 인수 : "미승인"
       - IF( OR( 3월 매출액 >= 50000, 총매출액 >= 150000), "승인", "미승인" )

- <span style="color:dodgerblue"> 조건문 </span>
    1. 합격여부
        - 조건 : 반전체 평균 이상이면 합격
        - IF( ) 1번만 사용
        - 반전체 평균은 움직이면 안되기 때문에 고정시켜 줘야함
        - 조건 인수
          - 평균 >= $(반전체평균)
        - VT 인수 : "합격", VF 인수 = "불합격"
        - IF( 평균 >= $(반전체평균), "합격", "불합격" )
          - 아래로 드래그하면 상대참조되서 다른 값들의 평균을 구할 수 있음
    
    2. 학점
        - 조건 : 국어성적이 80점 이상이면서 평균이 80점 이상인 학생은 "A" 그렇지 않으면 "B"
        - IF( ) 1번만 사용
        - 조건 인수
          - <span style="color:paleturquoise"> AND( ) </span> 사용
          - AND( 인수1, 인수2, ... , 인수n )
           - 인수 전부 부합하면 참
           - AND( 국어성적 >= 80, 평균 >= 80 )
        - VT 인수 : "A", VF 인수 : "B"
        - IF( AND( 국어성적 >= 80, 평균 >= 80 ), "A", "B" )
          - 아래로 드래그하면 상대참조되서 다른 값들의 평균을 구할 수 있음

    3. 재수강여부
        - 조건 : 국어성적이 70점 이상이거나 영어성적이 60점 이상이면 "PASS" 그렇지 않으면 "재수강"
        - IF( ) 1번만 사용
        - 조건 인수
          - OR( ) 사용
            - OR( 국어성적 >= 70, 영어성적 >= 60 )
        - VT 인수 : "PASS", VF 인수 : "재수강"
        - IF( OR( 국어성적 >= 70, 영어성적 >= 60 ), "PASS", "재수강" )
          - 아래로 드래그하면 상대참조되서 다른 값들의 평균을 구할 수 있음
